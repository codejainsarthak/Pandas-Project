# Pandas Practice Projects Repository

## Overview

This repository contains a collection of hands-on practice projects built using **Pandas and NumPy**. The purpose of this repository is to develop strong data analysis skills required for entry-level data analyst roles.

The projects focus on real-world scenarios such as student performance analysis, employee analytics, and sales data analysis.

---

## Objectives

* Build a strong foundation in Pandas
* Practice real-world data cleaning and transformation
* Understand data analysis workflows
* Prepare for data analyst interviews
* Develop project-based learning instead of theory-only learning

---

## Projects Included

### 1. Student Performance Analyzer

* Handling missing values
* Creating new columns (Total, Average)
* Pass/Fail classification using conditions
* Risk detection using `.where()`
* Sorting and filtering

---

### 2. Employee Analytics

* Data cleaning (handling missing salary)
* Merging datasets (employees + departments)
* Creating bonus and experience level columns
* GroupBy analysis:

  * Average salary per department
  * Total salary per department
  * Maximum salary per department
* Identifying top-performing employees

---

### 3. Sales & Orders Analytics

* Handling missing values in price
* Calculating total revenue per order
* Merging orders with product data
* Categorizing orders (High / Medium / Low)
* GroupBy analysis:

  * Sales per product
  * Sales per customer
  * Monthly sales trends
* Identifying highest-performing product and month

---

## Concepts Covered

* DataFrame creation and manipulation
* Data cleaning (`fillna`)
* Conditional operations (`np.where`, `.where`)
* Column operations and feature engineering
* Sorting and filtering
* Merging datasets (`pd.merge`)
* GroupBy and aggregation (`sum`, `mean`, `max`)
* Finding top values (`idxmax`)

---

## Tools Used

* Python
* Pandas
* NumPy

---

## How to Use This Repository

1. Clone the repository
2. Run each project file individually
3. Try solving problems independently before checking results
4. Modify datasets and experiment with different scenarios

---

## Learning Approach

This repository follows a practice-first approach:

* Focus on solving problems instead of memorizing syntax
* Learn by building small projects
* Improve through debugging and iteration

---

## Conclusion

This repository demonstrates a structured approach to learning Pandas through practical projects. It serves as a strong foundation for progressing toward more advanced data analysis and real-world applications.
